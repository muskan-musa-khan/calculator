# WinForms Calculator

A simple desktop calculator built using C# and Windows Forms. The application provides a clean interface for performing basic arithmetic operations and handles user input through an easy to use button layout.

### Features

- Perform addition, subtraction, multiplication, and division

- Clear and responsive button layout

- Handles decimal values

- Reset and clear functions

- Clean Windows Forms interface

### Built With

- C#

- Windows Forms

## Setup Instructions

### Requirements

- Visual Studio

- .NET Framework

### Clone the repository
git clone https://github.com/muskan-musa-khan/calculator

### Open the project

- Open the solution file in Visual Studio.

### Run the application

- Press the Start button in Visual Studio to launch the calculator.
